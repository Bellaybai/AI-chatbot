
### A full-stack AI chatbot using ChatGPT API and created with React, Node.js 
# Dependencies
## Backend
- ChatGPT
- Node.js
- Express
- RestfulAPI
- Postman
## Frontend
- React
- Vite

# Installation
## Backend
```bash
npm init -y
```
```bash
npm i openai cors body-parser express
```
```bash
touch index.js
```

## Frontend
```bash
npm create vite@latest
```
```bash
choose react, javascript
```
